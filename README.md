# Movers And Packers

A web-based Movers and Packers Management System designed to manage bookings, logistics, vehicles, staff, pricing, payments, and real-time tracking.

## Features
- Customer booking management
- Price estimation
- Vehicle and staff assignment
- Payment management
- Real-time tracking
- Admin dashboard
- Feedback and ratings

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- Maps: Google Maps API
- Payments: Stripe

## Folder Structure
Movers-And-Packers/
│── API/

## Installation
1. Clone the repository
2. Install backend dependencies
3. Install frontend dependencies
4. Run backend and frontend servers
